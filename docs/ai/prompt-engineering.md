# Prompt Engineering aplicado al desarrollo

En este documento guardo los prompts más útiles que he usado
durante el desarrollo, explicando por qué funcionan bien.

## 1. Prompt con rol definido
**Prompt:** "Actúa como un desarrollador senior y revisa este código
JavaScript buscando posibles errores o mejoras"
**Por qué funciona:** Definir un rol hace que la IA responda con
más profundidad técnica y menos explicaciones básicas.

## 2. Prompt para explicar conceptos
**Prompt:** "Explícame qué es un closure en JavaScript con un ejemplo
práctico sencillo"
**Por qué funciona:** Pedir un ejemplo práctico obliga a la IA a
ser concreta en vez de dar una explicación teórica abstracta.

## 3. Prompt para detectar bugs
**Prompt:** "Encuentra el error en este código y explícame por qué falla"
**Por qué funciona:** Es directo y concreto, la IA sabe exactamente
qué buscar sin ambigüedades.

## 4. Prompt paso a paso
**Prompt:** "Explícame paso a paso cómo funciona localStorage en JavaScript"
**Por qué funciona:** Pedir razonamiento paso a paso hace que la IA
estructure mejor la respuesta y no se salte pasos importantes.

## 5. Prompt con restricciones
**Prompt:** "Explícame qué es el DOM en máximo 5 líneas"
**Por qué funciona:** Las restricciones de longitud obligan a la IA
a priorizar lo más importante y ser concisa.

## 6. Prompt para generar código
**Prompt:** "Crea una función JavaScript que reciba un array de tareas
y devuelva solo las completadas. Usa filter y explica cada línea"
**Por qué funciona:** Especificar el método a usar y pedir explicación
garantiza código legible y educativo.

## 7. Prompt para refactorizar
**Prompt:** "Refactoriza esta función para que sea más legible y añade
comentarios JSDoc"
**Por qué funciona:** Dar dos objetivos concretos evita respuestas
vagas y asegura un resultado útil.

## 8. Prompt para documentar
**Prompt:** "Genera la documentación JSDoc para esta función JavaScript"
**Por qué funciona:** Es específico sobre el formato de documentación
que se quiere, evitando formatos incorrectos.

## 9. Prompt few-shot
**Prompt:** "Dado este ejemplo: input=[1,2,3] output=[2,4,6].
Crea una función que haga lo mismo con cualquier array"
**Por qué funciona:** Dar un ejemplo concreto ayuda a la IA a
entender exactamente qué se espera sin ambigüedades.

## 10. Prompt para comparar herramientas
**Prompt:** "Compara ChatGPT y Claude para tareas de desarrollo
web. Dame ventajas y desventajas de cada uno"
**Por qué funciona:** Pedir una comparación estructurada obliga
a la IA a ser objetiva y equilibrada en su respuesta.
